## English description

LineageProof turns DataHub MCP context into reviewable schema-change evidence before a migration ships. It reads entity, schema, lineage, query, and sensitive-field context, then produces deterministic findings, a remediation plan, and hash-only audit receipts.

The demo uses synthetic data only. It shows safe read checks, explicitly authorized write-back preview, blocking conditions, and reproducible outputs. It does not connect to a production account or contain customer data.

Public source: `https://github.com/heyhengl/lineageproof`
